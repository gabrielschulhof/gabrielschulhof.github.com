Detailed Description

Persistent storage handlers.

An APP must provide OCPersistentStorage handler pointers when it calls OCRegisterPersistentStorageHandler. Persistent storage open handler points to default file path. Application can point to appropriate SVR database path for it's IoTivity Server.
